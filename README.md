# Bookstore-SQL-project-2

End-to-end SQL project for an OnlineBookstore database using PostgreSQL.

### Overview
Designed a 3-table relational database and wrote 20+ queries for business analysis including sales tracking, inventory management, and customer segmentation.

### Database Schema
`Books` | `Customers` | `Orders` — with primary and foreign key relationships

### Key Queries in SQL_PROJECT_2.sql
- **Customer Analysis**: Top spending customer, high-value cities > $30
- **Sales KPIs**: Total revenue, books sold by genre, best-selling authors  
- **Inventory**: Remaining stock after orders, lowest stock alerts
- **Filtering**: By genre, date range, price, country

### Skills Demonstrated
`JOIN` `GROUP BY` `HAVING` `Aggregate Functions` `Foreign Keys` `Subqueries`

### Tech Stack
PostgreSQL, pgAdmin 4
